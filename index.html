HTML Table

Table Tags

Tag Name Description
<table> Table Represents tabular data
<th> Table Header Defines a cell as the header of a group of table cells
<tr> Table Row Defines a row in a table
<td> Table Data Cell Defines a cell of a table that contains data
<thead> Table Head Defining the head of the columns of the table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Foot Defines a set of rows summarizing the columns of the table
<caption> Table Caption Element specifies the caption (or title) of a table
<colgroup> Table Column Group Defines a group of columns within a table
<col> Table Column Element Specifies column properties for each column within a <colgroup> elemente

Table Attributes

Tag Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span
